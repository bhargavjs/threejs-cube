# Three.js Cube Project

This is a basic 3D cube project built using Three.js.

## Features
- 3D rotating cube
- Lighting (Point Light + Ambient Light)
- Smooth animation using requestAnimationFrame
- Responsive canvas

## Technologies Used
- HTML
- JavaScript
- Three.js

## How to Run
1. Download or clone the repository
2. Open index.html in browser

## Future Improvements
- Add mouse controls (OrbitControls)
- Add interaction (click events)
- Add multiple objects
